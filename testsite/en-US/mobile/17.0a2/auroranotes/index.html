Notes (First offered to Aurora channel users on August 31, 2012)

Check out "What’s New" and "Known Issues" for this version of Firefox for mobile below. As always, you’re encouraged to tell us what you think, or file a bug in Bugzilla.

What’s New

  • NEW
    Android 4/4.1: hardware and software decoder support for h.264 video
  • NEW
    Initial web app support
  • NEW
    Support for Android 4.1's Explore by Touch accessibility feature
  • DEVELOPER
    JavaScript Maps and Sets are now iterable
  • DEVELOPER
    SVG FillPaint and StrokePaint implemented
  • DEVELOPER
    Support for @supports added
  • HTML5
    Sandbox attribute for iframes implemented, enabling increased security

Known Issues

  • Unresolved
    Some recent changes in the Google layout and CSS may cause issues in non-webkit browsers like Firefox (see 668288)
  • Unresolved
    Firefox for Android uses the same locale as your system language settings (see 695565)
  • Unresolved
    Font inflation my sometimes render inconsistent font sizes on certain pages (see 707195)
  • Unresolved
    Changing the device orientation may sometimes yield unexpected behavior (see 740691)
  • Unresolved
    Firefox Sync will not behave as expected if multiple Firefox channels are installed on a single device (see 761206)
  • Unresolved
    Holding backspace may delete text both in front of and behind the cursor (see 770291)
